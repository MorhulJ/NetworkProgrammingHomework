#### Task 9:



1. Тому що це loopback-адреса, яка зарезервована для внутрішньої роботи комп’ютера, всі IP з діапазону 127.0.0.0/8 автоматично повертаються назад у ту ж саму систему, незалежно від мережі
2. У loopback інтерфейсу немає MAC адреси, тому що він не використовує мережеву карту, дані не передаються фізично через Ethernet або Wi-Fi, а обробляються всередині операційної системи
3. Ні, вони обробляються всередині TCP/IP стеку операційної системи і одразу повертаються назад
4. 127.0.0.1 — це loopback адреса, яка означає цей самий компютер. 0.0.0.0 — це всі IP-адреси або невизначена адреса, використовується для привязки сервера до всіх мережевих інтерфейсів або як початкове значення
